GPUOthello2
====

GPU Accelerated endgame solver for Othello
solve 6x6 board version

## Usage

### Prepare

```
$ mkdir value6x6
$ wget https://drive.google.com/file/d/1-F-y8YbwOIC3jsFn1GBBat4y-FzHkYqN/view?usp=sharing -o value6x6/value16_b
```

### INPUT

Input file path

#### Input file format

```
n
board 1
board 2
...
board n
```

each board format is base81
for detail: https://github.com/primenumber/issen

### OUTPUT

Output file path

### DEPTH

Maximum search depth (without pass), equal to maximum number of empty position
